MSSPM_GuiEstimation
This project contains the MSSPM Estimation GUIs.
Authors
Ronald Klasky
Date
Dec. 23, 2019

Description

The classes in this package define GUI forms available to the user for Estimation data input. In addition to data input, they allow the user to select the type of Estimation desired from various algorithms and sub-algorithms, along with its associated parameters. Finally, the user runs the Estimation from the final GUI, having output automatically directed to other GUIs in the application.
