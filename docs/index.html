Multi-Species Surplus Production Model (MSSPM)
Authors: Klasky, R., Townsend, Link. J., H., Gamble R., Beet A., and E. Wurst
The Multi-Species Surplus Production Model (MSSPM) allows the user to 1) define a model, using choices of growth, harvest, competition, and predation forms, 2) run the model, which estimates the necessary parameters using choices for optimization algorithms, and 3) visualize the estimated data.
